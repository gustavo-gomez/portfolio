# 🎨 My Personal Portfolio Website 🚀

Welcome to my personal portfolio website! 🎉 This project is designed to showcase my work, skills, and achievements in a sleek and interactive way. I hope you enjoy exploring it as much as I enjoyed building it!

- Next.js: A powerful React framework for server-side rendering and static site generation. It helps make the site fast and SEO-friendly. 🌐

- Tailwind CSS: A utility-first CSS framework that lets me build custom designs quickly and efficiently. 🎨
- next-intl: For adding internationalization, making the site accessible in multiple languages. 🌍
- TypeScript: Adds static typing to JavaScript, which helps in writing safer and more maintainable code. 💻
- GitHub Actions: Automates the deployment process with CI/CD, so updates are always smooth and automatic. 🔄
- Docker: Containerizes the app, ensuring it runs consistently across different environments. 📦
- Google Cloud Platform (GCP):
  - Cloud Run: Deploys and scales Docker containers effortlessly in the cloud. ☁️
  - Artifact Registry: Manages and stores Docker images for efficient deployment. 🛢️
- Google Analytics: Provides insights into website traffic and user interactions, helping me understand how people engage with the site. 📊

## ✨ Features

- Responsive Design: The site looks great on any device, from phones to desktops. 📱💻
- Internationalization: Available in multiple languages to reach a global audience. 🌐
- Continuous Deployment: Thanks to GitHub Actions, the latest changes are automatically tested and deployed. 🔄
- Analytics: Google Analytics helps track visitor behavior and site performance. 📈

The app is automatically deployed using GitHub Actions. Docker images are built and pushed to GCP Artifact Registry, and then deployed to Google Cloud Run. 🌟
